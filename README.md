# 🌳 BaobabAI

Africa's first continental AI model — trained on African languages, 
built for African businesses.

## Languages Supported
- Hausa | Igbo | Yoruba | Nigerian Pidgin
- Swahili | Amharic | Somali
- Xhosa | Lingala | Luganda

## Dataset
12,469 articles across 10 African languages (Phase 1)

## Vision
BaobabAI is the AI infrastructure layer for Africa — 
the same way AWS is the cloud layer for the world.
